## File Structure Documentation

* **`app/`**
  * `index.tsx` — Main frontend view
* **`src/`** — Backend source
  * **`config/`**
    * `firebase.js` — API keys & connection to Firebase
  * **`services/`**
    * `authService.js` — Authentication engine

## How to Run

**1. Install dependencies**

```bash
npm install
```

**2. Start the app**

```bash
npx expo start
```
